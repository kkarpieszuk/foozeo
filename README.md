# Foozeo Theme

Foozeo is a WordPress theme.

## Features

- Child theme of Twenty Twenty-Five
- Adds custom post type for books
- Adds custom taxonomy for genres
- Customizable FAQ accordion block which reuses core/accordion-item block (DRY!!!)

## How to install

### Like a pro:

1. Navigate to `wp-content/themes` directory on your server
2. Run `git clone git@github.com:kkarpieszuk/foozeo.git`
3. `cd foozeo && composer install`
4. Go to WordPress admin panel and activate the theme

### Easy mode:

1. Go to [Releases](https://github.com/kkarpieszuk/foozeo/releases)
2. Download the latest release
3. Upload the theme to your WordPress installation
4. Go to WordPress admin panel and activate the theme
